TERMUX

  1. 安装proot获取用户权限:

    pkg install proot

  2. 安装EMACS编辑器:

    pkg install emacs

  3. 安装OPENSSH服务器连接工具

    pkg install openssh

  4. 设置终端字体颜色:

    sh -c "$(curl -fsSL https://github.com/Cabbagec/termux-ohmyzsh/raw/master/install.sh)"

    ~/.termux/colors.sh

  5. 设置导航键:

    setting in file ".termux/termux.properties"

    extra-keys = [['ESC','TAB','CTRL','ALT','LEFT','UP','DOWN','RIGHT']]
